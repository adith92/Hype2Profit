# Hype2Profit Roadmap

## Phase 1
- Shopee-first market cockpit
- Tokopedia secondary support
- TikTok Shop placeholder only
- Visible-only Chrome extension
- Product scanner, competitor radar, watchlist, CSV export
- Premium mock fallback and legal notice

## Phase 2
- Auth, per-user watchlists, persisted exports
- Richer filters, saved scans, realtime alerts
- CSV import and supplier cost upload
- TikTok Shop deeper support when safe and justified

## Phase 3
- Official API integrations where allowed
- Benchmark cohorts, category dashboards, portfolio analytics
- Automated weekly intelligence brief

## Phase 4
- Collaborative workspaces
- Scenario planning, margin simulation, supply chain overlay
- Alert automation and deeper forecasting

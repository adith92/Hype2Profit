# Hype2Profit PRD

## Vision
Hype2Profit adalah "TradingView for Marketplace Product Research Indonesia" untuk membantu seller membaca momentum produk, peluang profit, dan risk signal lebih cepat dari kompetitor.

## Problem
- Riset produk masih tersebar di banyak tool dengan UI lama.
- Operator butuh kombinasi momentum trend, monetisasi, dan risk scan dalam satu panel.
- Banyak user tidak punya akses API resmi sehingga butuh jalur fallback yang tetap etis dan visible-only.
- Pasar kompetitif menuntut diferensiasi visual dan workflow, bukan clone scraper generik.

## Users
- Marketplace seller
- Brand incubator
- Category analyst
- Performance marketer

## Core Jobs
- Memantau produk yang sedang hype
- Menyaring produk dengan potensi margin
- Menganalisis toko kompetitor
- Menyimpan watchlist dan ekspor CSV
- Menyerap visible data dari extension tanpa bypass proteksi

## Phase 1 Scope
- Shopee-first dashboard dark-premium cockpit
- Tokopedia secondary visible DOM support
- TikTok Shop placeholder only
- Scanner, competitors, trending basic, watchlist, exports, settings, pricing placeholder
- Mock adapters berkualitas tinggi
- Supabase schema + fallback mode
- CSV export
- Chrome extension MV3 visible DOM ingest
- Legal notice and data policy emphasis

## Won't V1
- Payment
- Social media radar
- TikTok Shop deep analytics
- Backend scraper farm
- AI creator matching
- Auto crawling massive pages

## Success Criteria
- Prototype lokal berjalan penuh
- UI premium dan responsif
- Data mock realistis Indonesia
- Chart candlestick + histogram aktif
- Watchlist dan export berfungsi
- Data policy eksplisit dan aman
- Positioning jelas: product momentum, not just product list

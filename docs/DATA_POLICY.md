# Hype2Profit Data Policy

- Gunakan API resmi marketplace bila tersedia dan diizinkan.
- Hype2Profit bukan scraper backend agresif.
- Hype2Profit bukan bot checkout.
- Chrome extension hanya membaca data yang terlihat pada halaman yang sedang dibuka user.
- Hype2Profit tidak bypass proteksi marketplace, captcha, login, endpoint privat, rate limit, atau anti-bot.
- Hype2Profit tidak mengambil data personal pembeli.
- Hype2Profit tidak melakukan auto checkout, auto purchase, atau spam request.
- Hype2Profit tidak melakukan auto-scroll agresif untuk memanen halaman besar.
- Mock/demo adapter dipakai untuk prototype, development, dan fallback aman.
- CSV/manual upload adalah jalur fallback resmi untuk data milik user sendiri.
- Semua angka sales, revenue, views, momentum, dan risk signal pada V1 adalah estimasi.
- User bertanggung jawab mematuhi Terms of Service masing-masing platform.
